CONTROLS:
-MOUSE POSITION: controls drawing position, color, and direction
-BACKSPACE/DELETE: Clears the canvas
-SPACE: freezes current drawing
-SHIFT: resets the growth of the drawing position
-ENTER: downloads current image
-X: changes color palette randomly
-1: line drawing mode
-2: point drawing mode
-LEFT CLICK: modulo drawing mode
-MIDDLE CLICK: stops drawing while button is held
